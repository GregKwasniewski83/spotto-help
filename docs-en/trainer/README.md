# Trainer Screen

**Trainer panel in Spotto**: create your profile, conduct sessions, establish partnerships with sports facilities.

## Screen Overview

The Trainer screen is the hub for managing your coaching activity. You can create a profile, establish partnerships with sports facilities, create trainings, and manage participants.

![Trainer screen overview: Screenshot #38](../../assets/images/trainer/trainer-screen-overview.png)

### What You'll Find on the Trainer Screen

| Section | Description | More |
|---------|-------------|------|
| Trainer profile | Your details, specializations, rates | [Details](./features/trainer-profile.md) |
| Associated businesses | List of businesses you work with | [Details](./features/business-associations.md) |
| Trainings | Your sessions, participants, schedules | [Details](./features/training-management.md) |
| Schedule | Hourly availability for each day | [Details](./features/availability-schedule.md) |
