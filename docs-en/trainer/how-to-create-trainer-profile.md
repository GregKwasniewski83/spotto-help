# How to Create a Trainer Profile

**Screen**: Trainer
**Role**: Trainer
**Difficulty**: Medium

## Overview

Your trainer profile is your business card in Spotto. It contains contact details, specializations, experience, and rates. You can operate as an independent trainer or as a facility employee.

---

## Before You Start

- You must have a Spotto account with the "Trainer" role enabled (turn it on in [profile settings](../profile/how-to-manage-roles.md))
- If you operate as an independent trainer: prepare your tax ID (NIP), REGON, and business details

---

## Steps

### Step 1: Open the Trainer Screen

On the bottom navigation bar, tap the **Trainer** icon. If you don't have a profile yet, you will see the creation form.

### Step 2: Choose Your Trainer Type

| Type | Description |
|------|-------------|
| **Independent** | Self-employed: business details and TPay required |
| **Employee** | Working for a facility: simplified form |

![Trainer type selection: Screenshot #39](../../assets/images/trainer/trainer-type-selection.png)

> 💡 **Tip**: The "Employee" type requires less data, but you cannot create trainings on your own: the business does it for you.

### Step 3: Fill in Basic Details

| Field | Description | Required |
|-------|-------------|----------|
| **Display name** | Name/nickname visible to clients | Yes |
| **Email** | Contact email address | Yes |
| **Phone** | Contact number | Yes |
| **City** | Your city | Yes |
| **Address** | Business address | Yes |
| **Postal code** | Postal code | Yes |

![Trainer basic details: Screenshot #40](../../assets/images/trainer/trainer-basic-info.png)

### Step 4: (Independent) Fill in Business Details

For independent trainers only:

| Field | Description | Required |
|-------|-------------|----------|
| **NIP** | 10-digit tax identification number | Yes |
| **Company name** | Business name | Yes |
| **REGON** | 9 or 14 digits | No |
| **KRS** | 10 digits | No |
| **Specializations** | Select from the list (e.g., Squash, Shooting) | Yes |
| **Description** | Description of your practice | Yes |
| **Years of experience** | Coaching experience | No |
| **Certificates** | List of certificates (comma-separated) | No |
| **Languages** | Languages you conduct sessions in | No |

### Step 5: Set Your Hourly Rate

- **Net rate**: price per hour excluding VAT
- **VAT rate**: choose: 0%, 8%, or 23%
- **Gross rate**: calculated automatically

![Hourly rate: Screenshot #41](../../assets/images/trainer/trainer-hourly-rate.png)

### Step 6: (Independent) Fill in TPay Details

Independent trainers must configure payment details:

| Field | Required |
|-------|----------|
| **Legal form** | Yes |
| **Category (MCC)** | Yes |
| **Website** | Yes |
| **Contact person first name** | Yes |
| **Contact person last name** | Yes |

### Step 7: Add an Avatar

Tap the camera icon to add a profile photo:
- Take a photo with the camera
- Choose from gallery
- The photo will be cropped to a square format

### Step 8: Save the Profile

Tap **Save**. Your trainer profile will be created.

---

## Verification

The profile has been created when:
- You see a header with your name, city, and type label (Independent/Employee)
- The profile details card is visible
