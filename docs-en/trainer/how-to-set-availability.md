# How to Set Up an Availability Schedule

**Screen**: Trainer
**Role**: Trainer
**Difficulty**: Easy

## Overview

The availability schedule informs businesses and clients when you are free. You can set a general weekly schedule and override it for specific dates.

---

## Steps

### Step 1: Open the Schedule

On the Trainer screen, find the **Availability Schedule** section.

### Step 2: Set the General Schedule

Configure availability for three day types:

| Type | Description |
|------|-------------|
| **Weekdays** | Monday through Friday |
| **Saturday** | Saturday |
| **Sunday** | Sunday |

Select the hours (from 00:00 to 23:00) during which you are available.

![Trainer weekly schedule: Screenshot #48](../../assets/images/trainer/trainer-weekly-schedule.png)

### Step 3: (Optional) Set Availability for a Specific Date

1. Select the **Specific date** mode
2. Choose a date from the calendar
3. Select the availability hours

A schedule for a specific date has higher priority than the general schedule.

### Step 4: (Optional) Assign Hours to Businesses

If you work with multiple businesses, you can assign specific hours to specific businesses. This way, businesses can see when you are available for them.

### Step 5: Save the Schedule

Tap **Save** to confirm the changes.
