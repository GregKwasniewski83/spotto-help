# Trainer Profile

## Description

The trainer profile is your business card in Spotto. It contains contact details, specializations, experience, and hourly rates. Depending on the profile type (independent or employee), different data is required.

## Trainer Types

| Type | Description | Required Data |
|------|-------------|---------------|
| **Independent** | You run your own business | Business details (NIP, REGON, KRS), specializations, description, TPay data |
| **Employee** | You work for a sports facility | Basic details (name, contact, city) |

## Profile Data

### Basic Details (both types)

- Display name, email, phone
- City, address, postal code
- Avatar (profile photo)

### Business Details (independent only)

- NIP (10 digits), company name, REGON, KRS
- Specializations (multiple selection)
- Business description
- Years of experience, certificates, languages

### Hourly Rate

- Net rate (excluding VAT)
- VAT rate (0%, 8%, 23%)
- Gross rate (calculated automatically)

### TPay Data (independent only)

Legal form, MCC category, website, contact person details.

## Screen Header

At the top of the Trainer screen, you can see:
- Avatar
- Display name
- Type label (Independent / Employee)
- City
