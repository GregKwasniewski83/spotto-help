# How to Create a Training

**Screen**: Trainer
**Role**: Trainer
**Difficulty**: Medium

## Overview

A training is a session that you conduct at a sports facility. You can create trainings with multiple sessions on different dates, set prices, maximum participant count, and link the training to a facility reservation.

---

## Before You Start

- You must have a [trainer profile](./how-to-create-trainer-profile.md)
- You must have a [confirmed business partnership](./how-to-associate-with-business.md) with the "Own trainings" permission
- You need an active facility reservation

> ⚠️ **Important**: Trainers with "Employee" status cannot create trainings on their own: trainings are created by the business.

---

## Steps

### Step 1: Open the "Trainings" Tab

On the Trainer screen, go to the **Trainings** tab and tap **Create training**.

### Step 2: Fill in Training Details

| Field | Description | Required |
|-------|-------------|----------|
| **Title** | Training name (3-100 characters) | Yes |
| **Description** | Detailed description (10-1000 characters) | Yes |
| **Specialization** | Sport discipline | Yes |
| **Duration** | 15-480 minutes (in 15-min increments) | Yes |
| **Max. participants** | 1-50 people | Yes |
| **Net price** | Price per training (max 10,000 PLN) | Yes |
| **VAT rate** | 0%, 8%, or 23% | Yes |
| **Gross price** | Calculated automatically | : |

![Training creation form: Screenshot #44](../../assets/images/trainer/training-form.png)

### Step 3: Select a Facility Reservation

From the list of your upcoming reservations, select the facility where the training will take place. The list shows:
- Facility and business name
- Reservation dates
- Available time slots

![Reservation selection: Screenshot #45](../../assets/images/trainer/select-reservation.png)

### Step 4: Set Session Dates and Times

A training can have multiple sessions:
1. Select a date from the calendar
2. Set the start and end time
3. (Optionally) Add more dates

> 💡 **Tip**: You cannot select dates that have already passed. The end time must be later than the start time.

![Session configuration: Screenshot #46](../../assets/images/trainer/training-sessions.png)

### Step 5: Save the Training

Tap **Save**. The training will be created and will appear on the trainings list with session count information.

---

## Editing a Training

To edit an existing training:
1. On the trainings list, tap the edit icon
2. Change the necessary details
3. Confirm the changes

> ⚠️ **Important**: Editing a training with enrolled participants requires confirmation: changes may affect participants.
