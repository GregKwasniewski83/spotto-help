# Reservations Screen

**Your reservation management hub**: browse upcoming visits, manage payments, and cancel reservations in one place.

## Screen Overview

The Reservations screen appears after tapping the calendar icon on the bottom navigation bar. Here you will find a complete overview of your sports activities.

![IMG_RES_001 : Reservations screen overview](../../assets/images/reservations/reservations-overview.png)

### What You Will Find on the Reservations Screen

| Section | Description | More |
|---------|-------------|------|
| Upcoming events | List of reservations and trainings sorted by date | [Details](./features/reservation-details.md) |
| Pending payments | Unfinished transactions requiring attention | [How to complete](./how-to-pay-pending.md) |
| Reservation statuses | Color-coded reservation status indicators | [Details](./features/booking-status.md) |
| Cancellation | Full or partial cancellation with refund option | [Refund policy](./features/refund-policy.md) |
