# Product Catalog

## Description

The Product Catalog is the main part of the "Shop" tab. It displays products offered by sports facilities registered in Spotto: passes, entry packages, and subscriptions.

## Product Card

Each product is displayed as a card with the following information:

| Element | Description |
|---------|-------------|
| **Title** | Product name (e.g., "Monthly gym pass") |
| **Subtitle** | Additional description (optional) |
| **Price** | Gross price (including VAT) in PLN |
| **Number of uses** | How many times you can use the product (e.g., "10 uses") |
| **Validity period** | How long the product is active (e.g., "3 Months") |
| **Business** | Name of the sports facility offering the product |
| **"Buy Now" button** | Start the product purchase |
| **Share icon** | Send a product link to a friend |

![Product card elements: Screenshot #16](../../../assets/images/shop/product-card-anatomy.png)

## Product Types

Products vary by validity period:

| Period | Description | Example |
|--------|-------------|---------|
| **Days** | Valid for a specific number of days | 7-day pass |
| **Weeks** | Valid for a specific number of weeks | 2-week package |
| **Months** | Valid for a specific number of months | 3-month subscription |
| **Years** | Valid for a specific number of years | Annual pass |
| **Lifetime** | No expiration date | Unlimited pass |

## Products and Facilities

Products can be linked to:
- **All business facilities**: pass valid at all locations
- **Specific facilities**: pass valid only at selected locations

## Product Applications

Some products have additional properties:
- **Payable in-app**: can be used to pay for reservations directly in Spotto
- **Payable to trainer**: can be used to pay for training sessions with a trainer
