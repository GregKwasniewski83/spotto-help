# Product Search and Filters

## Description

The Spotto Shop offers a text search bar and a filter panel to help you find the right product among offerings from many sports facilities.

## Search Bar

The search field at the top of the Shop screen lets you search by:
- Product name
- Product description
- Business name (sports facility)
- City

Results filter automatically after typing a phrase (with a short 0.5-second delay).

## Filter Panel

After tapping the filter icon, a panel opens with options:

| Filter | Type | Description |
|--------|------|-------------|
| **Business** | Text | Sports facility name |
| **City** | Text | City where the facility is located |
| **Minimum price** | Number | Lower price limit (PLN) |
| **Maximum price** | Number | Upper price limit (PLN) |
| **Period** | Selection | Days, Weeks, Months, Years, Lifetime |

## Sorting Options

| Option | Description |
|--------|-------------|
| **Newest** | Sorted by date added (newest first) |
| **Price: ascending** | From cheapest to most expensive |
| **Price: descending** | From most expensive to cheapest |
| **Name: A-Z** | Alphabetically by name |
| **Uses: ascending** | From fewest entries |
| **Uses: descending** | From most entries |

## Business Filter (automatic)

If you navigate to the Shop from a specific facility profile, products will be automatically filtered. At the top of the screen, you'll see a banner "Products from selected facility" with a button to remove the filter.
