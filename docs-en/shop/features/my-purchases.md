# My Purchases

## Description

The "My Purchases" tab on the Shop screen contains a list of all products you've bought. Each purchase has its own status, usage counter, and expiration date.

## Purchase Card

Each purchased product is displayed as a card with the following information:

| Element | Description |
|---------|-------------|
| **Title and subtitle** | Name of the purchased product |
| **Status** | Active, Expired, or Used Up (with colored label) |
| **Progress bar** | Visual representation of remaining entries vs. total |
| **Purchase date** | When you bought the product |
| **Expiration date** | Until when the product is active |
| **Business** | Sports facility offering the product |
| **"Use Product" button** | Generates a verification code (active products only) |

## Purchase Statuses

| Status | Color | Meaning |
|--------|-------|---------|
| **Active** | Green (#10B981) | Product is valid, you can use it |
| **Expired** | Yellow (#F59E0B) | Validity period has passed |
| **Used Up** (Depleted) | Red (#EF4444) | All entries have been consumed |

## Filtering Purchases

Above the purchases list, there are filter buttons:
- **All**: displays all purchases
- **Active**: only products with "Active" status
- **Inactive**: expired and used up products

## Progress Bar

The progress bar visually shows how many entries remain:
- **Full (green)**: all entries available
- **Partially filled**: some entries used
- **Empty (red)**: all entries consumed

## Verification Code

After tapping "Use Product", a window displays:
- Verification code (8 characters, uppercase letters)
- Product and business details
- Business contact information (phone, email, address)
- Remaining entries counter
- Purchase and expiration dates
