# Shop Screen

**Shop in the Spotto app**: buy passes, packages, and subscriptions for sports facilities.

## Screen Overview

The Shop screen allows you to browse and purchase products offered by sports facilities: passes, entry packages, and subscriptions. After purchase, products appear in the "My Purchases" tab, where you can use them.

![Shop screen overview: Screenshot #1](../../assets/images/shop/shop-screen-overview.png)

### What you'll find on the Shop screen

| Section | Description | More |
|---------|-------------|------|
| "Shop" tab | Browse and buy products | [Details](./features/product-catalog.md) |
| "My Purchases" tab | Your purchased products and their status | [Details](./features/my-purchases.md) |
| Product search | Search by name, business, or city | [Details](./features/search-filters.md) |
| Filters and sorting | Narrow results by price, period, business | [Details](./features/search-filters.md) |
| Payment system | Secure payments via TPay | [Details](./features/payments.md) |
