# How to Change the App Language

**Screen**: Profile > Settings
**Role**: All
**Difficulty**: Easy

## Overview

Spotto is available in two languages: Polish and English. You can change the language at any time.

---

## Steps

### Step 1: Open Settings

On the Profile screen, tap the **Settings** icon (gear icon).

### Step 2: Find the "Preferences" Section

In the **Preferences** section, find the **Language** option.

### Step 3: Choose a Language

Tap your preferred language:
- **Polski**: default
- **English**: English

The language change is instant: the entire app interface will switch to the selected language.
