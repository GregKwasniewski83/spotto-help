# How to Manage Privacy Settings

**Screen**: Profile > Settings > Privacy
**Role**: All
**Difficulty**: Easy

## Overview

Privacy settings allow you to control what data is collected and how it is used. You can enable and disable individual options independently.

---

## Steps

### Step 1: Open Settings

On the Profile screen, tap the **Settings** icon (gear icon) in the upper right corner.

### Step 2: Open "Privacy and Security"

In the Privacy and security section, tap **Privacy settings**.

![Privacy settings: Screenshot #54](../../assets/images/profile/privacy-settings.png)

### Step 3: Adjust Settings

Each setting has a toggle that you can turn on or off:

| Setting | Description |
|---------|-------------|
| **Analytics** | Collection of app usage data to improve the app |
| **Crash reports** | Automatic sending of error reports |
| **Location tracking** | Collection of location data for recommendations |
| **Data sharing** | Anonymous data sharing with partners |
| **Marketing emails** | Receiving information about promotions and updates |
| **Push notifications** | Notifications about reservations and updates |

### Step 4: (Optional) Reset Analytics Data

At the bottom of the screen, you'll find the **Reset analytics** option: it deletes all collected analytics data.

> ⚠️ **Important**: Resetting analytics data is irreversible.
