# Profile Screen and Settings

**Your Spotto account**: manage personal data, roles, privacy, and app settings.

## Screen Overview

The Profile screen is your account management hub. Here you'll find your personal data, profile types (User, Business, Trainer), avatar, and access to privacy and app settings.

![Profile screen overview: Screenshot #49](../../assets/images/profile/profile-screen-overview.png)

### What You'll Find on the Profile Screen

| Section | Description | More |
|---------|-------------|------|
| Personal data | First name, last name, date of birth, phone | [Details](./features/account-settings.md) |
| Avatar | Your profile picture | [Details](./features/account-settings.md) |
| Interests | Your favorite sports activities | [Details](./features/account-settings.md) |
| Profile types | Roles: User, Business, Trainer | [Details](./features/role-management.md) |
| Location | Your current location | [Details](./features/account-settings.md) |

### What You'll Find in Settings

| Section | Description | More |
|---------|-------------|------|
| Language | Polish / English | [Details](./features/account-settings.md) |
| Privacy and security | Privacy settings, privacy policy | [Details](./features/privacy-settings.md) |
| Help and support | Contact the support team | : |
| About the app | Company details, app version | : |
| Log out | Sign out of your account | : |
