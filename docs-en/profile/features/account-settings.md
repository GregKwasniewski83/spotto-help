# Account Settings

## Description

Account settings allow you to manage your personal data, avatar, location, and language preferences.

## Personal Data

| Field | Description | Editable |
|-------|-------------|----------|
| **First name** | Your first name | Yes |
| **Last name** | Your last name | Yes |
| **Phone** | Contact number | Yes |
| **Date of birth** | Date of birth | Yes |
| **Interests** | Favorite sports (multiple selection) | Yes |

## Avatar

Spotto offers a set of animal avatars. Your avatar is visible:
- In the Profile screen header
- In comments and social posts
- In training participant lists

## Location

On the Profile screen, you can:
- See your current location (if permission is granted)
- Enable location access (button)
- Location is used to display nearby facilities

## App Settings

### Language
- **Polski** (default)
- **English**

The change is instant.
