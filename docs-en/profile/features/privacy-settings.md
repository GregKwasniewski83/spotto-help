# Privacy Settings

## Description

Privacy settings allow you to control what data the Spotto app collects and how it is used. All settings can be changed independently.

## Available Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Analytics** | Collection of usage data to improve the app | Enabled |
| **Crash reports** | Automatic sending of error reports | Enabled |
| **Location tracking** | Location data for nearby facility recommendations | Disabled |
| **Data sharing** | Anonymous data shared with partners | Disabled |
| **Marketing emails** | Information about promotions and updates | Disabled |
| **Push notifications** | Notifications about reservations and updates | Enabled |

## Resetting Analytics

The **Reset analytics** option deletes all collected analytics data. This operation is irreversible.

## Privacy Policy (GDPR)

The Spotto app processes personal data in accordance with GDPR. The full privacy policy is available in Settings > Privacy and security > Privacy policy.

### Collected Data

- Personal data (first name, last name, email, phone)
- Company data (NIP: for business roles)
- Reservation history
- Location data (with consent)

### Processing Purposes

- Maintaining the user account
- Processing reservations
- Handling complaints
- Marketing (with consent)
- Tax obligations

### Retention Period

- Tax data: 5 years
- Other data: until account deletion

### Your Rights

- Access to data
- Data correction
- Data deletion
- Data portability
- Objection to processing
