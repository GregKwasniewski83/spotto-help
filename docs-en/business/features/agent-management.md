# Agent Management

## Description

Agents are people with access to manage your business in Spotto. They can view bookings, manage facilities, and handle customers on your behalf. The system allows you to invite new agents by email, track invitation statuses, and revoke access when needed.

## How the Invitation System Works

The entire process of adding an agent consists of four stages:

1. **Invitation**: you enter the email address of the person you want to invite and send the invitation
2. **Pending**: the invitation appears on the "Pending Invitations" list. An email with an acceptance link is sent to the provided address
3. **Acceptance**: the person clicks the link in the email and becomes an agent of your business
4. **Expiration**: if the invitation is not accepted within a specified time, it expires automatically

![Agent section: Screenshot #56](../../../assets/images/business/agent-management-section.png)

> 💡 **Tip**: You can send invitations to multiple people: each will receive a separate email.

## Active Agents

After accepting an invitation, the agent appears on the **Active Agents** list. For each agent you can see:

| Information | Description |
|-------------|-------------|
| **Name** | Agent's data from their Spotto profile |
| **Email address** | Email used for the invitation |
| **Assignment date** | When the agent accepted the invitation |
| **Invited by** | Who sent the invitation |

### Removing an Agent

To revoke an agent's access:
1. Find the agent on the active list
2. Tap the **Remove** button
3. Confirm the removal in the dialog box

> ⚠️ **Important**: After removal, the agent immediately loses access to business management. The operation is irreversible: to restore access, you must send a new invitation.

## Pending Invitations

The pending invitations list shows people who have not yet accepted the invitation:

| Information | Description |
|-------------|-------------|
| **Email address** | Which address the invitation was sent to |
| **Invitation date** | When the invitation was sent |
| **Invited by** | Who sent it |
| **Status** | Active or Expired |

### Available Actions

- **Resend**: sends the invitation again (useful when the email went to spam or the person missed the message)
- **Cancel**: withdraws the invitation, the link in the email stops working

> 💡 **Tip**: Expired invitations are marked with a special label. You can resend them.

## Validation

The system checks the email address validity before sending an invitation. You cannot invite:
- A person who is already an agent
- A person with an active, pending invitation
- An invalid email address
