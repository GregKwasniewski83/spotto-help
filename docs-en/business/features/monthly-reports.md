# Monthly Reports

## Description

Monthly reports give you a complete picture of your business's financial and operational performance for a selected month. You can analyze revenue, number of bookings, cancellations, and a detailed breakdown by individual sports facilities.

## Report Types

### Summary Report

A quick overview of key metrics for the selected month:

| Metric | Description |
|--------|-------------|
| **Total revenue** | Sum of revenue from all facilities (in PLN) |
| **Total number of bookings** | How many bookings were completed |
| **Number of cancellations** | How many bookings were cancelled |
| **Facility breakdown** | Detailed data for each facility |

### Detailed Report

Contains everything from the summary, plus detailed lists:

| Section | Content |
|---------|---------|
| **Bookings** | List of individual bookings with dates, participants, and amounts |
| **Product purchases** | Pass and package purchase transactions with prices |
| **Trainer sessions** | List of trainer sessions with hours and rates |

## Facility Breakdown

In both report types you will find a table with the breakdown by individual facilities:

| Column | Description |
|--------|-------------|
| **Facility name** | Which sports facility |
| **Revenue** | Total revenue from this facility (PLN) |
| **Bookings** | Number of completed bookings |
| **Cancellations** | Number of cancelled bookings |

This allows you to see which facilities generate the most revenue and where the most cancellations occur.

## Period Selection

You can generate a report for any month:
- **Month**: select from the list (January -- December)
- **Year**: select from the available years (last 2-3 years)

> 💡 **Tip**: Generate reports regularly (e.g., at the end of each month) to track trends and make better business decisions.

## Export and Sharing

You can do the following with a generated report:
- **Download as PDF**: save to your device for archiving or accounting
- **Share**: send by email, messenger, or another app
- **Print**: print directly from your device

## Usage Examples

| Use Case | Which Report |
|----------|--------------|
| Quick monthly analysis | Summary |
| Accounting settlement | Detailed (PDF) |
| Facility comparison | Summary (breakdown table) |
| Trainer settlement | Detailed (trainer sessions) |
| Cancellation analysis | Summary (cancellation metric) |
