# Business Profile

## Description

The business profile is your business card in the Spotto app. It contains all the data needed to run your business: from basic information, through the schedule, to payment system integration data.

## Profile Data

### Basic Data

| Field | Description |
|-------|-------------|
| **NIP** | 10-digit tax identification number |
| **Company name** | Official name from the registry |
| **Display name** | Name visible to customers in the app |
| **Address** | Street, number, city, postal code |
| **Coordinates** | Location on the map (latitude/longitude) |
| **Email** | Contact email address |
| **Phone** | Contact phone number |

### Company Avatar

You can add a logo or company photo. The avatar is visible:
- On the company card in search results
- In the Business screen header
- In customers' booking details

### Schedule

The schedule defines business opening hours:
- **Weekdays**: Monday -- Friday
- **Saturday**: separate schedule
- **Sunday**: separate schedule
- **Specific dates**: schedule override for selected days

Each hourly slot (00:00 -- 23:00) can be: available, unavailable, or booked.

### Facility Floor Plan

You can upload a facility floor plan in the following formats:
- PDF
- Image (JPG, PNG)

The floor plan is visible to customers and helps with orientation within the facility.
