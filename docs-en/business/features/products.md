# Products (Passes and Packages)

## Description

Products are offers that you create for customers in the Spotto Shop. They can be passes, entry packages, or subscriptions with various prices, number of uses, and validity periods.

## Product Properties

| Property | Description |
|----------|-------------|
| **Title** | Name visible in the Shop |
| **Subtitle** | Additional description (optional) |
| **Description** | Detailed product description |
| **Net/gross price** | With automatic VAT conversion |
| **Number of uses** | How many times the customer can use it |
| **Period** | Days, Weeks, Months, Years, Lifetime |
| **Number of periods** | E.g., "3" months |
| **Active** | Whether the product is visible in the Shop |

## Payment Options

| Option | Description |
|--------|-------------|
| **Payable in app** | Customer uses the product to pay for a booking |
| **Payable at trainer** | Customer uses the product to pay for a training session |

## Facility Assignment

A product can be valid:
- **At all facilities** of the business
- **At selected facilities**: you select specific locations

## Time Restriction

Optionally, you can set:
- **Start date**: product available in the Shop from this date
- **End date**: product disappears from the Shop after this date
