# Business Screen Features

The Business screen is the management hub for your sports business in Spotto. Below you will find detailed descriptions of all available features: from basic profile configuration, through facility and staff management, to payment and invoicing system integrations.

## Company Configuration

| Feature | Description | Difficulty |
|---------|-------------|------------|
| [Business Profile](./business-profile.md) | Company data, avatar, contact details, schedule | Medium |
| [Facility Management](./facility-management.md) | Creating facilities, pricing, VAT, availability, floor plans | Medium |
| [TPay Integration](./tpay-integration.md) | Payment gateway configuration for customers | Advanced |
| [KSeF Integration](./ksef-integration.md) | Electronic invoicing (token or certificate) | Advanced |

## Offer and Sales

| Feature | Description | Difficulty |
|---------|-------------|------------|
| [Products](./products.md) | Creating passes, entry packages, and subscriptions | Medium |
| [Monthly Reports](./monthly-reports.md) | Revenue, booking, and cancellation analysis | Easy |

## Staff and Collaboration

| Feature | Description | Difficulty |
|---------|-------------|------------|
| [Agent Management](./agent-management.md) | Inviting employees, managing access | Easy |
| [Trainer Associations](./trainer-associations.md) | Accepting trainers, rates, permissions | Medium |
| [Child Businesses](./child-businesses.md) | Managing branches, TPay/NIP permissions | Advanced |
