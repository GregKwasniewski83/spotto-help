# How to Add a Sports Facility

**Screen**: Business
**Role**: Business Owner
**Difficulty**: Medium

## Overview

Sports facilities are the places you offer customers for booking: courts, shooting ranges, gyms, etc. Each facility has its own name, type, price, and availability schedule.

---

## Before You Start

- You must have a [business profile created](./how-to-create-profile.md)

---

## Steps

### Step 1: Open the "Facilities" Section

On the Business screen, find the **Sports Facilities** section and tap the **Add Facility** button.

![Facilities section: Screenshot #21](../../assets/images/business/facilities-section.png)

### Step 2: Fill in Facility Data

Complete the facility form:

| Field | Description | Required |
|-------|-------------|----------|
| **Name** | Facility name (e.g., "Squash Court #1") | Yes |
| **Type** | Facility type (from dropdown list) | Yes |
| **Description** | Facility description for customers | No |
| **Address** | Street and number of the facility | Yes |
| **City** | City | Yes |
| **Postal code** | Postal code | Yes |

![Facility form: Screenshot #22](../../assets/images/business/facility-form.png)

### Step 3: Set the Price

Configure the hourly rate:
- **Net price**: price without VAT
- **VAT rate**: choose from the list: 0%, 5%, 8%, 23%, or EX (exempt)
- **Gross price**: calculated automatically based on the net price and VAT rate

> **Tip**: You can also enter the gross price: the net price will be calculated automatically.

![Price configuration: Screenshot #23](../../assets/images/business/facility-pricing.png)

### Step 4: Set Additional Parameters

| Parameter | Description |
|-----------|-------------|
| **Minimum number of slots** | Minimum booking duration (e.g., 1 hour) |
| **Maximum number of users** | Facility capacity (how many people at once) |
| **Price per person** | Whether the price is charged per person (toggle) |

### Step 5: Configure Availability

Select the hours during which the facility is available for booking:
- Choose hourly slots (from 00:00 to 23:00)
- Selected = available, deselected = unavailable

### Step 6: (Optional) Upload a Facility Plan

You can add a facility plan in PDF format or as a photo so customers can see the layout.

### Step 7: Save the Facility

Tap **Save**. The facility will appear on the list in the "Sports Facilities" section.

---

## Verification

The facility was added correctly when:
- You can see it on the facilities list on the Business screen
- Customers can find it in the search on the Home screen

---

## Troubleshooting

### Problem: I can't save the facility
**Solution**: Check that all required fields (name, type, address) are filled in and that the price is a valid number.
