# I Can't Log In

## Problem

You cannot log in to the Spotto app : you see an error or the login screen doesn't accept your credentials.

## Most Common Causes and Solutions

### Wrong password

**Symptoms**: Message "Invalid email or password"

**Solution**:
1. Check that Caps Lock is not enabled
2. Make sure you are entering the correct email address
3. If you don't remember your password : [reset it](./forgot-password.md)

### Account doesn't exist

**Symptoms**: Message about a non-existent account

**Solution**:
1. Check if the registration was completed
2. Make sure you are using the same email address as during registration
3. If the account doesn't exist, register again

### No internet connection

**Symptoms**: Loading never finishes, timeout

**Solution**:
1. Check Wi-Fi or mobile data connection
2. Open a browser and check if other websites work
3. Switch between Wi-Fi and mobile data

### Too many failed attempts

**Symptoms**: Message about temporary lock

**Solution**:
1. Wait a few minutes and try again
2. If the problem persists : [reset your password](./forgot-password.md)

---

## Next Steps

If the above solutions didn't help:
- Make sure you have the latest version of the app
- Uninstall and reinstall the app
- Contact support@spotto.pl
