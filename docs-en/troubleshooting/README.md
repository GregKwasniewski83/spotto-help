## Quick Fixes

Before looking for a detailed solution, try these steps:

1. **Check your internet connection** : most features require internet
2. **Close and reopen the app** : fixes many temporary issues
3. **Update the app** : make sure you have the latest version
4. **Restart your phone** : helps with memory issues
