# Sport Categories

## Description

The sport categories section (Popular Activities) is an interactive carousel on the Home screen that lets you quickly browse venues by sporting discipline.

## Available Categories

![Sport categories carousel on Home screen](../../../assets/images/home/feat-categories-carousel.png)

| Category | Icon | Description |
|----------|------|-------------|
| Shooting | 🎯 | Sports and recreational shooting ranges |
| Squash | 🏸 | Squash courts |
| Gym | 💪 | Weight rooms and fitness centers |
| Tennis | 🎾 | Tennis courts (indoor and outdoor) |
| Padel | 🏓 | Padel courts |
| Football | ⚽ | Pitches (community, indoor, full-size) |
| Swimming pool | 🏊 | Swimming pools and aquaparks |
| Badminton | 🏸 | Badminton courts |
| Table tennis | 🏓 | Table tennis tables |
| Climbing | 🧗 | Climbing walls |

> 💡 **Note**: The category list is dynamic and depends on the venues registered in the Spotto system. New categories appear automatically.

## How the Carousel Works

- **Scrolling** : swipe left/right to see more categories
- **Selection** : tap a category to see venues offering that discipline
- **Results** : after selecting a category, a list of venues is displayed (similar to search results)

![Category selected - facility results list](../../../assets/images/home/feat-category-results.png)

## Differences Between Search and Categories

| Aspect | Search | Categories |
|--------|--------|------------|
| Location selection | Manual | Automatic (based on location) |
| Date filter | Yes | No (shows general availability) |
| Type filter | Yes (dropdown list) | Yes (tap icon) |
| Speed | Requires typing | One tap |
