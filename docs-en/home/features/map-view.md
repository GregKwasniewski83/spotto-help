# Map view

## Description

The map view is an interactive section on the Home screen that displays sports facilities in your area. The map uses GPS location to show the nearest places on the map with markers.

## Map elements

| Element | Description |
|---------|-------------|
| **Blue dot** | Your current location |
| **Colored markers** | Sports facilities (pins on the map) |
| **Crosshair button** | Centers the map on your position |
| **+/- buttons** | Zooming in and out of the map |
| **Info window** | Appears after tapping a marker |

## Navigation gestures

- **Panning**: drag your finger across the map
- **Zooming in**: pinch with two fingers (pinch-to-zoom) or tap +
- **Zooming out**: spread two fingers apart or tap -
- **Double tap**: quick zoom in

## Interacting with markers

After tapping a marker, an info window appears with:
- Facility name
- Address
- Distance from your location
- User rating
- Buttons: **Details** and **Book**

## Location permissions

The map requires access to GPS location. Without it:
- The map still works, but it will not show your position
- Facilities will not be sorted by distance
- The crosshair button will be inactive

### How to enable location

**Android**: Settings → Location → Enable → Spotto → Allow

**iOS**: Settings → Privacy → Location Services → Spotto → While Using
