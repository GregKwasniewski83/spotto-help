# Facility search

## Description

The search tool is the main feature of the Home screen. It is located at the top of the screen and allows you to find sports facilities based on location, date, and facility type.

## Available filters

| Filter | Description | Required |
|--------|-------------|----------|
| **City/location** | City name or address | Yes |
| **Date** | Booking day (default: today) | No |
| **Facility type** | Type of facility (court, shooting range, gym...) | No |

## How the search works

1. **Autocomplete**: as you type a city name, you will see suggestions. Tap a suggestion to select it.
2. **Search**: after filling in the filters, tap the search button or confirm with the keyboard.
3. **Results**: a list of facilities sorted by relevance and distance.

## Result card

Each search result is displayed as a card with the following information:

- **Facility name**: the official name of the place
- **Address**: street, number, city
- **Type**: type of facility (e.g., "Squash court")
- **Availability**: available time slots for the selected day
- **Price**: approximate price per hour
- **Rating**: average user rating (stars)
- **Book button**: quick access to booking

## Sorting results

Results are sorted by relevance by default. Factors that affect the order:
- Match to the query
- Distance from the user's location
- Facility popularity
- Time slot availability

## Limitations

- The search requires an internet connection
- Results are limited to facilities registered in the Spotto system
- Autocomplete works for Polish cities
