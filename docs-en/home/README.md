# Home Screen (Start)

**The main screen of the Spotto app** : Your hub for searching and booking sports facilities.

## Screen Overview

The Home screen is the first page you see when you launch the app. Here you'll find all the tools you need to find and book the perfect sports facility.

![Home Screen Overview](../../assets/images/home/home-screen-overview.png)

### What you'll find on the Home screen

| Section | Description | More |
|---------|-------------|------|
| Search | Search for facilities by city, date, and type | [Details](./features/search.md) |
| Sport categories | Browse facilities by sport discipline | [Details](./features/categories.md) |
| Favorite facilities | Quick access to saved facilities | [Details](./features/favorites.md) |
| Nearby map | Sports facilities in your area | [Details](./features/map-view.md) |
| Upcoming reservations | Preview of your nearest bookings | [Reservations](../reservations/README.md) |
| Pending payments | Notifications about incomplete payments | [How to book](./how-to-book-facility.md) |
