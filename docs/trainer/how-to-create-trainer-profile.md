# Jak utworzyć profil trenera

**Ekran**: Trener
**Rola**: Trener
**Trudność**: Średnia

## Przegląd

Profil trenera to Twoja wizytówka w Spotto. Zawiera dane kontaktowe, specjalizacje, doświadczenie i stawki. Możesz działać jako trener niezależny lub jako pracownik obiektu.

---

## Zanim zaczniesz

- Musisz mieć konto w Spotto z aktywną rolą „Trener" (włącz ją w [ustawieniach profilu](../profile/how-to-manage-roles.md))
- Jeśli działasz jako trener niezależny : przygotuj NIP, REGON i dane firmy

---

## Kroki

### Krok 1: Otwórz ekran Trener

Na dolnym pasku nawigacji dotknij ikony **Trener**. Jeśli nie masz jeszcze profilu, zobaczysz formularz tworzenia.

### Krok 2: Wybierz typ trenera

| Typ | Opis |
|-----|------|
| **Niezależny** | Samodzielna działalność : wymagane dane firmy i TPay |
| **Pracownik** | Pracujesz dla obiektu : uproszczony formularz |

![Wybór typu trenera : Screenshot #39](../../assets/images/trainer/trainer-type-selection.png)

> 💡 **Wskazówka**: Typ „Pracownik" wymaga mniej danych, ale nie możesz samodzielnie tworzyć treningów : to robi firma.

### Krok 3: Wypełnij dane podstawowe

| Pole | Opis | Wymagane |
|------|------|----------|
| **Nazwa wyświetlana** | Imię/pseudonim widoczny dla klientów | Tak |
| **E-mail** | Kontaktowy adres e-mail | Tak |
| **Telefon** | Numer kontaktowy | Tak |
| **Miasto** | Twoje miasto | Tak |
| **Adres** | Adres działalności | Tak |
| **Kod pocztowy** | Kod pocztowy | Tak |

![Dane podstawowe trenera : Screenshot #40](../../assets/images/trainer/trainer-basic-info.png)

### Krok 4: (Niezależny) Wypełnij dane firmy

Tylko dla trenerów niezależnych:

| Pole | Opis | Wymagane |
|------|------|----------|
| **NIP** | 10-cyfrowy numer podatkowy | Tak |
| **Nazwa firmy** | Nazwa działalności | Tak |
| **REGON** | 9 lub 14 cyfr | Nie |
| **KRS** | 10 cyfr | Nie |
| **Specjalizacje** | Wybierz z listy (np. Squash, Strzelectwo) | Tak |
| **Opis** | Opis Twojej działalności | Tak |
| **Lata doświadczenia** | Staż trenerski | Nie |
| **Certyfikaty** | Lista certyfikatów (oddzielone przecinkami) | Nie |
| **Języki** | Języki, w których prowadzisz zajęcia | Nie |

### Krok 5: Ustaw stawkę godzinową

- **Stawka netto** : cena za godzinę bez VAT
- **Stawka VAT** : wybierz: 0%, 8% lub 23%
- **Stawka brutto** : oblicza się automatycznie

![Stawka godzinowa : Screenshot #41](../../assets/images/trainer/trainer-hourly-rate.png)

### Krok 6: (Niezależny) Wypełnij dane TPay

Trenerzy niezależni muszą skonfigurować dane do płatności:

| Pole | Wymagane |
|------|----------|
| **Forma prawna** | Tak |
| **Kategoria (MCC)** | Tak |
| **Strona internetowa** | Tak |
| **Imię osoby kontaktowej** | Tak |
| **Nazwisko osoby kontaktowej** | Tak |

### Krok 7: Dodaj avatar

Dotknij ikony aparatu, aby dodać zdjęcie profilowe:
- Zrób zdjęcie aparatem
- Wybierz z galerii
- Zdjęcie zostanie przycięte do formatu kwadratowego

### Krok 8: Zapisz profil

Dotknij **Zapisz**. Twój profil trenera zostanie utworzony.

---

## Weryfikacja

Profil został utworzony, gdy:
- Widzisz nagłówek z Twoim imieniem, miastem i etykietą typu (Niezależny/Pracownik)
- Karta z danymi profilu jest widoczna
