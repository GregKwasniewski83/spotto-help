# Widok mapy

## Opis

Widok mapy to interaktywna sekcja na ekranie Home, która wyświetla obiekty sportowe w Twojej okolicy. Mapa wykorzystuje lokalizację GPS, aby pokazać najbliższe miejsca na mapie z markerami.

## Elementy mapy

| Element | Opis |
|---------|------|
| **Niebieska kropka** | Twoja aktualna lokalizacja |
| **Kolorowe markery** | Obiekty sportowe (pinezki na mapie) |
| **Przycisk celownika** | Centrowanie mapy na Twojej pozycji |
| **Przyciski +/-** | Przybliżanie i oddalanie mapy |
| **Okienko informacyjne** | Pojawia się po dotknięciu markera |

## Gesty nawigacji

- **Przesuwanie** : przeciągnij palcem po mapie
- **Przybliżanie** : szczypnij dwoma palcami (pinch-to-zoom) lub dotknij +
- **Oddalanie** : rozsuń dwa palce lub dotknij -
- **Podwójne dotknięcie** : szybkie przybliżenie

## Interakcja z markerami

Po dotknięciu markera pojawia się okienko z:
- Nazwą obiektu
- Adresem
- Odległością od Twojej lokalizacji
- Oceną użytkowników
- Przyciskami: **Szczegóły** i **Rezerwuj**

## Uprawnienia lokalizacji

Mapa wymaga dostępu do lokalizacji GPS. Bez tego:
- Mapa nadal działa, ale nie pokaże Twojej pozycji
- Obiekty nie będą sortowane wg odległości
- Przycisk celownika będzie nieaktywny

### Jak włączyć lokalizację

**Android**: Ustawienia → Lokalizacja → Włącz → Spotto → Zezwól

**iOS**: Ustawienia → Prywatność → Usługi lokalizacyjne → Spotto → Podczas używania
