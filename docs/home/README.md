# Home Screen - Pomoc

**Ekran główny aplikacji Spotto**

## Przegląd

Ekran Home (Start) jest głównym ekranem aplikacji, gdzie możesz:
- Wyszukiwać obiekty sportowe
- Przeglądać kategorie sportów
- Zarządzać ulubionymi obiektami
- Wyświetlać obiekty na mapie
- Szybko rezerwować wybrane obiekty

## Główne funkcje

### Wyszukiwanie obiektów
Znajdź obiekt sportowy wpisując miasto, wybierając datę i typ obiektu.
- [Jak wyszukać obiekt](./how-to-search-facilities.md) *(do utworzenia)*

### Przeglądanie kategorii
Przejrzyj dostępne obiekty według sportu (Strzelectwo, Squash, Siłownia, itp.)
- [Jak przeglądać kategorie](./how-to-browse-categories.md) *(do utworzenia)*

### Ulubione obiekty
Zapisz swoje ulubione obiekty dla szybkiego dostępu.
- [Jak zarządzać ulubionymi](./how-to-manage-favorites.md) *(do utworzenia)*

### Widok mapy
Wyświetl obiekty w pobliżu na mapie.
- [Jak korzystać z mapy](./how-to-use-map.md) *(do utworzenia)*

### Rezerwacja
Zarezerwuj obiekt bezpośrednio z wyników wyszukiwania.
- [Jak zarezerwować obiekt](./how-to-book-facility.md) *(do utworzenia)*
