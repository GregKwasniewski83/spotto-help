# Ekran Home (Start)

**Ekran główny aplikacji Spotto** : Twoje centrum wyszukiwania i rezerwacji obiektów sportowych.

## Przegląd ekranu

Ekran Home to pierwsza strona, którą widzisz po uruchomieniu aplikacji. Znajdziesz tu wszystkie narzędzia potrzebne do znalezienia i zarezerwowania idealnego obiektu sportowego.

![Przegląd ekranu Home](../../assets/images/home/home-screen-overview.png)

### Co znajdziesz na ekranie Home

| Sekcja | Opis | Więcej |
|--------|------|--------|
| Wyszukiwarka | Szukaj obiektów po mieście, dacie i typie | [Szczegóły](./features/search.md) |
| Kategorie sportów | Przeglądaj obiekty wg dyscypliny sportowej | [Szczegóły](./features/categories.md) |
| Ulubione obiekty | Szybki dostęp do zapisanych obiektów | [Szczegóły](./features/favorites.md) |
| Mapa okolicy | Obiekty sportowe w Twojej okolicy | [Szczegóły](./features/map-view.md) |
| Nadchodzące rezerwacje | Podgląd najbliższych rezerwacji | [Rezerwacje](../reservations/README.md) |
| Oczekujące płatności | Powiadomienia o nieukończonych płatnościach | [Jak zarezerwować](./how-to-book-facility.md) |
| Tablica społeczności | Posty i aktywność innych użytkowników | [Szczegóły](./features/community-wall.md) |
