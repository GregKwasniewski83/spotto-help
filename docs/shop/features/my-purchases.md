# Moje zakupy

## Opis

Zakładka „Moje zakupy" na ekranie Sklep zawiera listę wszystkich produktów, które kupiłeś/aś. Każdy zakup ma swój status, licznik wykorzystania i datę ważności.

## Karta zakupu

Każdy zakupiony produkt wyświetla się jako karta z informacjami:

| Element | Opis |
|---------|------|
| **Tytuł i podtytuł** | Nazwa zakupionego produktu |
| **Status** | Aktywny, Wygasły lub Wykorzystany (z kolorową etykietą) |
| **Pasek postępu** | Wizualne przedstawienie pozostałych wejść vs. całkowitych |
| **Data zakupu** | Kiedy kupiłeś/aś produkt |
| **Data ważności** | Do kiedy produkt jest aktywny |
| **Firma** | Obiekt sportowy, który oferuje produkt |
| **Przycisk „Użyj produktu"** | Generuje kod weryfikacyjny (tylko dla aktywnych) |

## Statusy zakupów

| Status | Kolor | Znaczenie |
|--------|-------|-----------|
| **Aktywny** (Active) | Zielony (#10B981) | Produkt jest ważny, możesz go wykorzystać |
| **Wygasły** (Expired) | Żółty (#F59E0B) | Minął termin ważności |
| **Wykorzystany** (Depleted) | Czerwony (#EF4444) | Zużyłeś/aś wszystkie wejścia |

## Filtrowanie zakupów

Nad listą zakupów znajdują się przyciski filtrów:
- **Wszystkie** — wyświetla wszystkie zakupy
- **Aktywne** — tylko produkty ze statusem „Aktywny"
- **Nieaktywne** — wygasłe i wykorzystane produkty

## Pasek postępu

Pasek postępu wizualnie pokazuje, ile wejść zostało:
- **Pełny (zielony)** — wszystkie wejścia dostępne
- **Częściowo zapełniony** — część wejść wykorzystana
- **Pusty (czerwony)** — wszystkie wejścia zużyte

## Kod weryfikacyjny

Po dotknięciu „Użyj produktu" wyświetla się okno z:
- Kodem weryfikacyjnym (8 znaków, duże litery)
- Danymi produktu i firmy
- Informacjami kontaktowymi firmy (telefon, e-mail, adres)
- Licznikiem pozostałych wejść
- Datami zakupu i ważności
