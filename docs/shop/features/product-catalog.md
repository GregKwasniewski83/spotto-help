# Katalog produktów

## Opis

Katalog produktów to główna część zakładki „Sklep". Wyświetla produkty oferowane przez obiekty sportowe zarejestrowane w Spotto : karnety, pakiety wejść i abonamenty.

## Karta produktu

Każdy produkt wyświetla się jako karta z następującymi informacjami:

| Element | Opis |
|---------|------|
| **Tytuł** | Nazwa produktu (np. „Karnet miesięczny na siłownię") |
| **Podtytuł** | Dodatkowy opis (opcjonalny) |
| **Cena** | Cena brutto (z VAT) w PLN |
| **Liczba użyć** | Ile razy można skorzystać z produktu (np. „10 użyć") |
| **Okres ważności** | Jak długo produkt jest aktywny (np. „3 Miesiące") |
| **Firma** | Nazwa obiektu sportowego, który oferuje produkt |
| **Przycisk „Kup teraz"** | Rozpocznij zakup produktu |
| **Ikona udostępniania** | Wyślij link do produktu znajomemu |

## Rodzaje produktów

Produkty różnią się okresem ważności:

| Okres | Opis | Przykład |
|-------|------|---------|
| **Dni** | Ważny przez określoną liczbę dni | Karnet 7-dniowy |
| **Tygodnie** | Ważny przez określoną liczbę tygodni | Pakiet 2-tygodniowy |
| **Miesiące** | Ważny przez określoną liczbę miesięcy | Abonament 3-miesięczny |
| **Lata** | Ważny przez określoną liczbę lat | Karnet roczny |
| **Dożywotni** | Bez daty wygaśnięcia | Karnet bezterminowy |

## Produkty a obiekty

Produkty mogą być powiązane z:
- **Wszystkimi obiektami firmy** : karnet ważny we wszystkich lokalizacjach
- **Konkretnymi obiektami** : karnet ważny tylko w wybranych lokalizacjach

## Zastosowanie produktów

Niektóre produkty mają dodatkowe właściwości:
- **Płatne w aplikacji** : mogą być użyte do opłacenia rezerwacji bezpośrednio w Spotto
- **Płatne u trenera** : mogą być użyte do opłacenia zajęć z trenerem
