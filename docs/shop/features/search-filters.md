# Wyszukiwarka i filtry produktów

## Opis

Sklep Spotto oferuje wyszukiwarkę tekstową oraz panel filtrów, które pomagają znaleźć odpowiedni produkt spośród ofert wielu obiektów sportowych.

## Wyszukiwarka

Pole wyszukiwania na górze ekranu Sklep pozwala szukać po:
- Nazwie produktu
- Opisie produktu
- Nazwie firmy (obiektu sportowego)
- Mieście

Wyniki filtrują się automatycznie po wpisaniu frazy (z krótkim opóźnieniem 0,5 sekundy).

## Panel filtrów

Po dotknięciu ikony filtrów otwiera się panel z opcjami:

| Filtr | Typ | Opis |
|-------|-----|------|
| **Firma** | Tekst | Nazwa obiektu sportowego |
| **Miasto** | Tekst | Miasto, w którym znajduje się obiekt |
| **Cena minimalna** | Liczba | Dolna granica ceny (PLN) |
| **Cena maksymalna** | Liczba | Górna granica ceny (PLN) |
| **Okres** | Wybór | Dni, Tygodnie, Miesiące, Lata, Dożywotni |

## Opcje sortowania

| Opcja | Opis |
|-------|------|
| **Najnowsze** | Posortowane wg daty dodania (najnowsze pierwsze) |
| **Cena: rosnąco** | Od najtańszych do najdroższych |
| **Cena: malejąco** | Od najdroższych do najtańszych |
| **Nazwa: A-Z** | Alfabetycznie wg nazwy |
| **Użycia: rosnąco** | Od najmniejszej liczby wejść |
| **Użycia: malejąco** | Od największej liczby wejść |

## Filtr firmy (automatyczny)

Jeśli przejdziesz do Sklepu z profilu konkretnego obiektu, produkty zostaną automatycznie przefiltrowane. Na górze ekranu zobaczysz baner „Produkty wybranego obiektu" z przyciskiem do usunięcia filtra.
