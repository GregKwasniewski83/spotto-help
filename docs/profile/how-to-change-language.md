# Jak zmienić język aplikacji

**Ekran**: Profil → Ustawienia
**Rola**: Wszystkie
**Trudność**: Łatwa

## Przegląd

Spotto jest dostępne w dwóch językach: polskim i angielskim. Możesz zmienić język w dowolnym momencie.

---

## Kroki

### Krok 1: Otwórz Ustawienia

Na ekranie Profil dotknij ikony **Ustawienia** (koło zębate).

### Krok 2: Znajdź sekcję „Preferencje"

W sekcji **Preferencje** znajdź opcję **Język**.

### Krok 3: Wybierz język

Dotknij wybranego języka:
- **Polski** : domyślny
- **English** : angielski

Zmiana języka jest natychmiastowa : cały interfejs aplikacji przełączy się na wybrany język.
