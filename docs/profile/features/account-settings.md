# Ustawienia konta

## Opis

Ustawienia konta pozwalają zarządzać danymi osobowymi, avatarem, lokalizacją i preferencjami językowymi.

## Dane osobowe

| Pole | Opis | Edytowalne |
|------|------|------------|
| **Imię** | Twoje imię | Tak |
| **Nazwisko** | Twoje nazwisko | Tak |
| **Telefon** | Numer kontaktowy | Tak |
| **Data urodzenia** | Data urodzenia | Tak |
| **Zainteresowania** | Ulubione sporty (wielokrotny wybór) | Tak |

## Avatar

Spotto oferuje zestaw avatarów ze zwierzętami. Avatar jest widoczny:
- W nagłówku ekranu Profil
- W komentarzach i postach społecznościowych
- W listach uczestników treningów

## Lokalizacja

Na ekranie Profil możesz:
- Zobaczyć swoją aktualną lokalizację (jeśli pozwolenie jest udzielone)
- Włączyć dostęp do lokalizacji (przycisk)
- Lokalizacja jest używana do wyświetlania obiektów w pobliżu
