# Ustawienia prywatności

## Opis

Ustawienia prywatności pozwalają kontrolować, jakie dane zbiera aplikacja Spotto i jak są one wykorzystywane. Wszystkie ustawienia można zmieniać niezależnie.

## Dostępne ustawienia

| Ustawienie | Opis | Domyślnie |
|------------|------|-----------|
| **Analityka** | Zbieranie danych o użytkowaniu do ulepszania aplikacji | Włączone |
| **Raporty awarii** | Automatyczne wysyłanie raportów o błędach | Włączone |
| **Śledzenie lokalizacji** | Dane lokalizacyjne do rekomendacji obiektów w pobliżu | Wyłączone |
| **Udostępnianie danych** | Anonimowe dane udostępniane partnerom | Wyłączone |
| **E-maile marketingowe** | Informacje o promocjach i nowościach | Wyłączone |
| **Powiadomienia push** | Powiadomienia o rezerwacjach i aktualizacjach | Włączone |

## Resetowanie analityki

Opcja **Zresetuj analitykę** usuwa wszystkie zebrane dane analityczne. Ta operacja jest nieodwracalna.

## Polityka prywatności (RODO)

Aplikacja Spotto przetwarza dane osobowe zgodnie z RODO. Pełna polityka prywatności jest dostępna w Ustawieniach → Prywatność i bezpieczeństwo → Polityka prywatności.

### Zbierane dane

- Dane osobowe (imię, nazwisko, e-mail, telefon)
- Dane firmy (NIP — dla ról biznesowych)
- Historia rezerwacji
- Dane lokalizacyjne (za zgodą)

### Cele przetwarzania

- Prowadzenie konta użytkownika
- Realizacja rezerwacji
- Obsługa reklamacji
- Marketing (za zgodą)
- Obowiązki podatkowe

### Okres przechowywania

- Dane podatkowe: 5 lat
- Inne dane: do momentu usunięcia konta

### Twoje prawa

- Dostęp do danych
- Poprawianie danych
- Usunięcie danych
- Przenoszenie danych
- Sprzeciw wobec przetwarzania
