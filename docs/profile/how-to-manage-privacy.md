# Jak zarządzać ustawieniami prywatności

**Ekran**: Profil → Ustawienia → Prywatność
**Rola**: Wszystkie
**Trudność**: Łatwa

## Przegląd

Ustawienia prywatności pozwalają kontrolować, jakie dane są zbierane i jak są wykorzystywane. Możesz włączać i wyłączać poszczególne opcje niezależnie.

---

## Kroki

### Krok 1: Otwórz Ustawienia

Na ekranie Profil dotknij ikony **Ustawienia** (koło zębate) w prawym górnym rogu.

### Krok 2: Otwórz „Prywatność i bezpieczeństwo"

W sekcji Prywatność i bezpieczeństwo dotknij **Ustawienia prywatności**.

![Ustawienia prywatności — Screenshot #54](../../assets/images/profile/privacy-settings.png)

### Krok 3: Dostosuj ustawienia

Każde ustawienie ma przełącznik, który możesz włączyć lub wyłączyć:

| Ustawienie | Opis |
|------------|------|
| **Analityka** | Zbieranie danych o użytkowaniu aplikacji w celu jej ulepszania |
| **Raporty awarii** | Automatyczne wysyłanie raportów o błędach |
| **Śledzenie lokalizacji** | Zbieranie danych o lokalizacji do rekomendacji |
| **Udostępnianie danych** | Anonimowe udostępnianie danych partnerom |
| **E-maile marketingowe** | Otrzymywanie informacji o promocjach i nowościach |
| **Powiadomienia push** | Powiadomienia o rezerwacjach i aktualizacjach |

### Krok 4: (Opcjonalnie) Zresetuj dane analityczne

Na dole ekranu znajdziesz opcję **Zresetuj analitykę** — usuwa wszystkie zebrane dane analityczne.

> ⚠️ **Ważne**: Resetowanie danych analitycznych jest nieodwracalne.
