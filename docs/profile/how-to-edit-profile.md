# Jak edytować dane profilu

**Ekran**: Profil
**Rola**: Wszystkie
**Trudność**: Łatwa

## Przegląd

Na ekranie Profil możesz zmienić swoje dane osobowe: imię, nazwisko, numer telefonu, datę urodzenia i zainteresowania sportowe.

---

## Kroki

### Krok 1: Otwórz ekran Profil

Na dolnym pasku nawigacji dotknij ikony **Profil**.

### Krok 2: Dotknij „Edytuj profil"

Na karcie danych osobowych dotknij przycisku **Edytuj**. Otworzy się formularz edycji.

![Edycja profilu : Screenshot #50](../../assets/images/profile/edit-profile-form.png)

### Krok 3: Zmień dane

Możesz zmienić:

| Pole | Opis |
|------|------|
| **Imię** | Twoje imię |
| **Nazwisko** | Twoje nazwisko |
| **Telefon** | Numer kontaktowy |
| **Data urodzenia** | Twoja data urodzenia |
| **Zainteresowania** | Wybierz ulubione sporty (wielokrotny wybór) |

![Zainteresowania sportowe : Screenshot #51](../../assets/images/profile/interests-selection.png)

> 💡 **Wskazówka**: Zainteresowania pomagają aplikacji dopasowywać rekomendacje do Twoich preferencji.

### Krok 4: Zapisz zmiany

Dotknij **Zapisz**. Dane zostaną zaktualizowane natychmiast.
