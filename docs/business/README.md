# Ekran Biznes (Zarządzanie)

**Panel zarządzania firmą w Spotto** : twórz profil firmy, zarządzaj obiektami, produktami, trenerami i płatnościami.

## Przegląd ekranu

Ekran Biznes to centrum zarządzania Twoją firmą sportową. Znajdziesz tu wszystkie narzędzia potrzebne do prowadzenia działalności: od tworzenia profilu firmy, przez zarządzanie obiektami i harmonogramami, po integrację z systemami płatności i fakturowania.
