# Zarządzanie obiektami sportowymi

## Opis

Obiekty sportowe to miejsca (korty, strzelnice, siłownie itp.), które udostępniasz klientom do rezerwacji. Każdy obiekt ma swoją nazwę, typ, cenę, pojemność i harmonogram dostępności.

## Dane obiektu

| Pole | Opis |
|------|------|
| **Nazwa** | Nazwa obiektu (np. „Kort squashowy nr 1") |
| **Typ** | Rodzaj obiektu z listy predefiniowanych typów |
| **Opis** | Opis widoczny dla klientów |
| **Adres** | Ulica, miasto, kod pocztowy |
| **Cena netto** | Cena za godzinę bez VAT |
| **Stawka VAT** | 0%, 5%, 8%, 23% lub ZW (zwolniony) |
| **Cena brutto** | Obliczana automatycznie (netto + VAT) |
| **Min. liczba slotów** | Minimalna długość rezerwacji |
| **Maks. użytkowników** | Pojemność obiektu |
| **Cena za osobę** | Czy cena jest naliczana za osobę |

## Cennik i VAT

System automatycznie przelicza ceny:
- Wpisz cenę netto → brutto oblicza się automatycznie
- Wpisz cenę brutto → netto oblicza się automatycznie
- Zmiana stawki VAT automatycznie aktualizuje drugą cenę

Dostępne stawki VAT:

| Stawka | Opis |
|--------|------|
| **0%** | Stawka zerowa |
| **5%** | Stawka obniżona |
| **8%** | Stawka obniżona (usługi sportowe) |
| **23%** | Stawka podstawowa |
| **ZW** | Zwolniony z VAT |

## Harmonogram obiektu

Każdy obiekt ma własny harmonogram 24-godzinny (00:00 – 23:00). Sloty mogą mieć status:
- **Dostępny** : klient może zarezerwować
- **Zarezerwowany** : już zajęty
- **Niedostępny** : obiekt zamknięty

## Plan obiektu

Możesz dodać plan obiektu (PDF lub zdjęcie), aby klienci mogli zobaczyć układ miejsca.
