# Profil firmy

## Opis

Profil firmy to Twoja wizytówka w aplikacji Spotto. Zawiera wszystkie dane potrzebne do prowadzenia działalności: od podstawowych informacji, przez harmonogram, po dane do integracji z systemami płatności.

## Dane profilu

### Dane podstawowe

| Pole | Opis |
|------|------|
| **NIP** | 10-cyfrowy numer identyfikacji podatkowej |
| **Nazwa firmy** | Oficjalna nazwa z rejestru |
| **Nazwa wyświetlana** | Nazwa widoczna dla klientów w aplikacji |
| **Adres** | Ulica, numer, miasto, kod pocztowy |
| **Współrzędne** | Lokalizacja na mapie (szerokość/długość geograficzna) |
| **E-mail** | Kontaktowy adres e-mail |
| **Telefon** | Kontaktowy numer telefonu |

### Avatar firmy

Możesz dodać logo lub zdjęcie firmy. Avatar jest widoczny:
- Na karcie firmy w wynikach wyszukiwania
- W nagłówku ekranu Biznes
- W szczegółach rezerwacji klientów

### Harmonogram

Harmonogram określa godziny otwarcia firmy:
- **Dni robocze** : poniedziałek – piątek
- **Sobota** : osobny grafik
- **Niedziela** : osobny grafik
- **Konkretne daty** : nadpisanie grafiku na wybrane dni

Każdy slot godzinowy (00:00 – 23:00) może być: dostępny, niedostępny lub zarezerwowany.

### Plan obiektu

Możesz wgrać plan obiektu w formacie:
- PDF
- Zdjęcie (JPG, PNG)

Plan jest widoczny dla klientów i pomaga w orientacji w obiekcie.
