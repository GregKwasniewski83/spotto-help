# Produkty (karnety i pakiety)

## Opis

Produkty to oferty, które tworzysz dla klientów w Sklepie Spotto. Mogą to być karnety, pakiety wejść lub abonamenty z różnymi cenami, liczbą wejść i okresami ważności.

## Właściwości produktu

| Właściwość | Opis |
|------------|------|
| **Tytuł** | Nazwa widoczna w Sklepie |
| **Podtytuł** | Dodatkowy opis (opcjonalny) |
| **Opis** | Szczegółowy opis produktu |
| **Cena netto/brutto** | Z automatycznym przeliczaniem VAT |
| **Liczba użyć** | Ile razy klient może skorzystać |
| **Okres** | Dni, Tygodnie, Miesiące, Lata, Dożywotni |
| **Liczba okresów** | Np. „3" miesiące |
| **Aktywny** | Czy produkt jest widoczny w Sklepie |

## Opcje płatności

| Opcja | Opis |
|-------|------|
| **Płatny w aplikacji** | Klient używa produktu do opłacenia rezerwacji |
| **Płatny u trenera** | Klient używa produktu do opłacenia treningu |

## Przypisanie do obiektów

Produkt może być ważny:
- **We wszystkich obiektach** firmy
- **W wybranych obiektach** : zaznaczasz konkretne lokalizacje

## Ograniczenie czasowe

Opcjonalnie możesz ustawić:
- **Data rozpoczęcia** : produkt dostępny w Sklepie od tej daty
- **Data zakończenia** : produkt znika ze Sklepu po tej dacie
