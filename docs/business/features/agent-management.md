# Zarządzanie agentami

## Opis

Agenci to osoby z dostępem do zarządzania Twoją firmą w Spotto. Mogą przeglądać rezerwacje, zarządzać obiektami i obsługiwać klientów w Twoim imieniu. System pozwala zapraszać nowych agentów e-mailem, śledzić statusy zaproszeń i w razie potrzeby cofać dostęp.

## Jak działa system zaproszeń

Cały proces dodawania agenta składa się z czterech etapów:

1. **Zaproszenie** : wpisujesz adres e-mail osoby, którą chcesz zaprosić, i wysyłasz zaproszenie
2. **Oczekiwanie** : zaproszenie trafia na listę „Oczekujące zaproszenia". Na podany adres e-mail wysyłana jest wiadomość z linkiem do akceptacji
3. **Akceptacja** : osoba klika link w e-mailu i staje się agentem Twojej firmy
4. **Wygaśnięcie** : jeśli zaproszenie nie zostanie zaakceptowane w określonym czasie, wygasa automatycznie

> 💡 **Wskazówka**: Możesz wysłać zaproszenia do wielu osób : każda otrzyma osobny e-mail.

## Aktywni agenci

Po zaakceptowaniu zaproszenia agent pojawia się na liście **Aktywni agenci**. Dla każdego agenta widzisz:

| Informacja | Opis |
|------------|------|
| **Imię i nazwisko** | Dane agenta z jego profilu Spotto |
| **Adres e-mail** | E-mail użyty do zaproszenia |
| **Data przypisania** | Kiedy agent zaakceptował zaproszenie |
| **Zaproszony przez** | Kto wysłał zaproszenie |

### Usuwanie agenta

Aby cofnąć dostęp agentowi:
1. Znajdź agenta na liście aktywnych
2. Dotknij przycisku **Usuń**
3. Potwierdź usunięcie w oknie dialogowym

> ⚠️ **Ważne**: Po usunięciu agent natychmiast traci dostęp do zarządzania firmą. Operacja jest nieodwracalna : aby przywrócić dostęp, musisz wysłać nowe zaproszenie.

## Oczekujące zaproszenia

Lista oczekujących zaproszeń pokazuje osoby, które jeszcze nie zaakceptowały zaproszenia:

| Informacja | Opis |
|------------|------|
| **Adres e-mail** | Na jaki adres wysłano zaproszenie |
| **Data zaproszenia** | Kiedy zaproszenie zostało wysłane |
| **Zaproszony przez** | Kto wysłał |
| **Status** | Aktywne lub Wygasłe |

### Dostępne akcje

- **Wyślij ponownie** : wysyła zaproszenie ponownie (przydatne, gdy e-mail trafił do spamu lub osoba nie zauważyła wiadomości)
- **Anuluj** : wycofuje zaproszenie, link w e-mailu przestaje działać

> 💡 **Wskazówka**: Zaproszenia, które wygasły, są oznaczone specjalną etykietą. Możesz je wysłać ponownie.

## Walidacja

System sprawdza poprawność adresu e-mail przed wysłaniem zaproszenia. Nie można zaprosić:
- Osoby, która jest już agentem
- Osoby z aktywnym, oczekującym zaproszeniem
- Nieprawidłowego adresu e-mail
